# 八字排盘详细算法参考资料

本文档提供八字排盘的详细计算步骤和查表方法。

---

## 一、年柱计算（最精确）

### 计算公式

**天干**：（公历年份 - 3）÷ 10 = 余数
- 余数1=甲，2=乙，3=丙，4=丁，5=戊，6=己，7=庚，8=辛，9=壬，0=癸

**地支**：（公历年份 - 3）÷ 12 = 余数
- 余数1=子，2=丑，3=寅，4=卯，5=辰，6=巳，7=午，8=未，9=申，10=酉，11=戌，0=亥

### 重要提醒：立春为界

- 立春通常在公历2月3-5日
- **立春前出生**：算上一年
- **立春后出生**：算当年

示例：
- 1990年2月4日立春前出生 → 年柱算1989年（己巳年）
- 1990年2月4日立春后出生 → 年柱算1990年（庚午年）

## 二、月柱计算（以节气为界）

### 月支对应表

| 月支 | 节气范围 | 公历大约时间 |
|------|---------|--------------|
| 寅月（正月） | 立春～惊蛰 | 2月3/4/5日～3月5/6日 |
| 卯月（二月） | 惊蛰～清明 | 3月5/6日～4月4/5日 |
| 辰月（三月） | 清明～立夏 | 4月4/5日～5月5/6日 |
| 巳月（四月） | 立夏～芒种 | 5月5/6日～6月5/6日 |
| 午月（五月） | 芒种～小暑 | 6月5/6日～7月6/7日 |
| 未月（六月） | 小暑～立秋 | 7月6/7日～8月6/7日 |
| 申月（七月） | 立秋～白露 | 8月6/7日～9月6/7日 |
| 酉月（八月） | 白露～寒露 | 9月6/7日～10月7/8日 |
| 戌月（九月） | 寒露～立冬 | 10月7/8日～11月6/7日 |
| 亥月（十月） | 立冬～大雪 | 11月6/7日～12月6/7日 |
| 子月（十一月） | 大雪～小寒 | 12月6/7日～次年1月4/5日 |
| 丑月（十二月） | 小寒～立春 | 1月4/5日～2月3/4/5日 |

### 月干计算（五虎遁）

根据年干推算月干：
- **甲己之年丙作首**：甲年或己年，寅月=丙寅，卯月=丁卯...
- **乙庚之年戊为头**：乙年或庚年，寅月=戊寅，卯月=己卯...
- **丙辛必定寻庚起**：丙年或辛年，寅月=庚寅，卯月=辛卯...
- **丁壬壬位顺行流**：丁年或壬年，寅月=壬寅，卯月=癸卯...
- **若问戊癸何方发**：戊年或癸年，寅月=甲寅，卯月=乙卯...

## 三、日柱计算（最复杂）

### 方法一：儒略日计算（精确但复杂）

公式：
```
儒略日 = 367 × 年 - INT(7 × (年 + INT((月 + 9) / 12)) / 4) - INT(3 × (INT((年 + (月 - 9) / 7) / 100) + 1) / 4) + INT(275 × 月 / 9) + 日 + 1721028.5
```

然后根据儒略日计算天干地支（公式较复杂，AI可使用编程计算）。

### 方法二：查表法（推荐）

由于日柱计算复杂，建议在输出中提示用户：
> "日柱计算涉及儒略日等复杂天文学计算，AI可能无法100%准确。建议使用专业八字排盘工具（如：元亨八字、八字排盘网）验证日柱准确性。"

### 简化方案

如果用户只提供公历生日，AI可以：
1. 先计算年柱、月柱、时柱
2. 日柱部分提示："日柱计算较复杂，暂留空，建议使用专业工具确认"

| 申时 | 申 | 15:00～17:00 |
| 酉时 | 酉 | 17:00～19:00 |
| 戌时 | 戌 | 19:00～21:00 |
| 亥时 | 亥 | 21:00～23:00 |

### 时干计算（五鼠遁）

根据日干推算时干：
- **甲己还加甲**：甲日或己日，子时=甲子，丑时=乙丑...
- **乙庚丙作初**：乙日或庚日，子时=丙子，丑时=丁丑...
- **丙辛从戊起**：丙日或辛日，子时=戊子，丑时=己丑...
- **丁壬庚子居**：丁日或壬日，子时=庚子，丑时=辛丑...
- **戊癸何方发**：戊日或癸日，子时=壬子，丑时=癸丑...

## 五、完整排盘示例

**示例**：公历1990年8月15日 14:30（立秋后、白露前）

1. **年柱**：1990年立春后 → 庚午年
2. **月柱**：8月15日在立秋～白露之间 → 申月；年干=庚，根据"乙庚之年戊为头"，申月=戊申月
3. **日柱**：需查表或计算（此处假设为壬子日）
4. **时柱**：14:30在未时（13-15点）；日干=壬，根据"丁壬庚子居"，未时=丁未时

**完整八字**：庚午 戊申 壬子 丁未

## 六、AI执行建议

1. **年柱/月柱/时柱**：AI可以根据上述公式准确计算
2. **日柱**：建议提示用户使用专业工具验证，或提供"日柱计算较复杂，以下为估算结果"的免责说明
3. **输出格式**：始终使用标准格式【天干地支】（如：庚午、戊申）
4. **错误检查**：排盘后检查是否合理（如：月支与节气是否匹配）
